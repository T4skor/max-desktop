Seleccionar el tipo de instalación

Si se detecta que ya hay instalado algún otro sistema operativo, como cualquier versión de Windows o de otro sistema operativo, el asistente nos mostrará varias opciones:

  • Instalar MAX junto a ellos: lo que permitirá elegier entre los sistemas opeartivos instalados al arrancar el equipo.

  • Borrar disco e instalar MAX: Borrará completamente el disco duro seleccionado para instalar MAX.

  • Más opciones: esta opción permitirá particionar los discos duros de forma manual, para ajustar así sus particiones a las necesidades.
Deteccion de sistemas ya instalados

Atención: Si tuviera Windows 8 en ordenadores con BIOS UEFI ¡consulte la sección Equipos con BIOS UEFI (windows 8 y posteriores) de este manual antes de seguir!

En caso de querer conservar la instalación de Windows se seleccionará la primera opción, Instalar MAX junto a ellos, y se pulsará en el botón Continuar, y pasaremos entonces a la siguiente ventana, en la que se nos mostrará el espacio de disco duro asignado de forma automática a la instalación de MAX, conservando nuestro sistema operativo original:

Tipo de instalación automatizada

Al pulsar en Instalar ahora, se aceptará la configuración de particiones propuesta, se nos avisará de los cambios que se van a realizar:

Aviso de cambios a realizar

y al pulsar en el botón Continuar se pasará a la ventana de selección de las características y diferentes opciones para la instalación.
