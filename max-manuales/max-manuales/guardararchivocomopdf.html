Guardar archivo como PDF

Si se quiere generar un documento que los alumnos u otros compañeros puedan leer, pero no modificar, se puede generar un archivo PDF con el contenido. El formato de archivo PDF está completamente extendido y es estándar, por lo que no tendrá problema alguno de visualización en ningún sitio, sin importar el sistema operativo desde el que se vea.

LibreOffice permite exportar cualquier tipo de documento que generemos (archivos de writer, hojas de cálculo, presentaciones, etc) en formato PDF.

Para ello, siga el sencillo paso que se muestra en la imagen siguiente:

Para modificar alguna de las opciones de creación del fichero PDF seleccione la opción de "Archivo → Exportar como pdf" y seleccionar el nombre y la ubicación del archivo a generar.
